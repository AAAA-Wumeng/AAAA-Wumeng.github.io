查

查询的语法有查询之后返回 QuerySet 的查询，比如 filter()，exclude()

也有 返回单个 object 的查询，比如 get()

对于 QuerySet，这个我们可以简单理解为是多个 object 实例形成的列表，但是这个列表是Django的一种特有的形式，具有能进行其他条件筛选的功能。

filter()，过滤筛选，返回的是符合条件的数据

exclude()，排除筛选，返回的是不符合条件的数据
